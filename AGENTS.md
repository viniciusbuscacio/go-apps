# go-apps — agent notes

This hub repo hosts the family specification: **[`go-apps.spec`](go-apps.spec)**.

**Rule number one: read `go-apps.spec` before working on any repo of the
family.** It is the single source of truth for architecture, UI rules,
updater/release contracts and working conventions. UI/UX patterns with
assets live in the go-design repo, which the spec delegates to.

New rules and learnings go INTO the spec (bump its version + changelog) —
not into this file. This AGENTS.md, like the one in every family repo, is
only the pointer that gets agents to the spec.
